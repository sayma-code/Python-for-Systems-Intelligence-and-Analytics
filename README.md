*This repository has all the task with their respective number in the course. The project folder is the final Project. The texts below is describing the project.*

This project demonstrates a complete analytics workflow:
ETL → Cleaning → Visualization → Prediction → Prescription → Benchmarking → BI Reporting
It provides actionable insights into how economic, environmental, and social factors shape child mortality outcomes globally.

**Data Sources**

1. PDF – Created from WHO data include density and literacy (via tabula-py)
2. SQL (MySQL) – Collected from WHO Child Mortality API data
3. Excel – Collected from World Bank GDP & Health Expenditure
4. CSV – AQI (Air Quality Index) dataset
