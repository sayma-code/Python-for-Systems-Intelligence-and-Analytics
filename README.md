This project demonstrates a complete analytics workflow:
ETL → Cleaning → Visualization → Prediction → Prescription → Benchmarking → BI Reporting
It provides actionable insights into how economic, environmental, and social factors shape child mortality outcomes globally.

#Data Sources

PDF – WHO Air Quality Report (via tabula-py)
SQL (MySQL) – WHO Child Mortality API data
Excel – World Bank GDP & Health Expenditure
CSV – AQI (Air Quality Index) dataset
